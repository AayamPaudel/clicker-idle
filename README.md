# Clicker Idle
This is a web-based clicking game. It is a game made with HTML, CSS and JavaScript. Click to earn money, upgrade your income per click, and unlock autoclicker to earn money faster.

### Features
1) Click to earn money
2) Upgrade to earn more per click
3) Buy autoclicker to make money automatically
4) Upgrade prices as you progress
5) Simple and clean UI
   
### How to play
1) Click the "Make money" Button
2) Use money to upgrade click power
3) Buy autoclicker to earn money automatically
